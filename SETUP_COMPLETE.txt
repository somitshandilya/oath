ðŸŽ¯ SETUP STEPS (UI-ONLY)
================================================================================

Step 1: Enable Module (2 minutes)
  - Go to Manage > Extend
  - Search for "simple oauth"
  - Check the box
  - Click Install

Step 2: Configure Settings (2 minutes)
  - Go to Manage > Configuration > Development > OAuth2 Server
  - Set Public Key Path: ../keys/public.key
  - Set Private Key Path: ../keys/private.key
  - Click Save

Step 3: Create OAuth Client (3 minutes)
  - Go to Manage > Structure > OAuth2 Clients
  - Click Add OAuth2 Client
  - Fill in Label and Redirect URI
  - Click Save
  - Copy Client ID and Secret

Step 4: Test (3 minutes)
  - Navigate to authorization endpoint
  - Log in and grant permission
  - Receive authorization code

Total Time: ~10 minutes

ðŸ” IMPORTANT SECURITY NOTES
================================================================================

1. SAVE YOUR CREDENTIALS SECURELY
   - Client ID: Can be shared
   - Client Secret: KEEP SECRET! (shown only once)
   - Save in password manager or secure location

2. USE HTTPS IN PRODUCTION
   - These guides use http://localhost for development
   - Always use https:// in production

3. VALIDATE REDIRECT URIs
   - Only add trusted URLs
   - Must match exactly (including protocol and port)

4. PROTECT YOUR PRIVATE KEY
   - Never commit to version control
   - Ensure web server can read it
   - Restrict file permissions
